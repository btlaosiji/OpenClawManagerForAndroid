OpenClaw 管理 用户协议

欢迎使用 OpenClaw 管理（下称「本应用」）。使用本应用即表示您同意以下条款。

一、服务说明

本应用为 OpenClaw Gateway 的移动端管理工具，主要功能包括：

  • 连接您自行部署的 OpenClaw Gateway（通过 WebSocket）；
  • 与 Gateway 进行对话、查看状态监控、配置模型与 API Key、搜索并复制 Skill 安装命令等。

本应用不提供云端账号或托管服务，所有连接与数据均在您指定的 Gateway 与设备之间进行。

二、使用条件

您需自行部署或有权使用 OpenClaw Gateway，并保证：

  • 遵守当地法律法规及 Gateway 与第三方服务（如模型 API、ClawHub 等）的使用条款；
  • 妥善保管 Gateway 地址与 Token，避免泄露给未授权方。

三、免责声明

本应用按「现状」提供。因网络环境、Gateway 配置、第三方服务变更等导致的连接失败、功能异常或数据问题，本应用不承担责任。您使用本应用与 Gateway 及第三方服务产生的任何后果由您自行承担。

四、条款变更

我们可能适时更新本协议，更新后的协议将在应用内或发布渠道公布。继续使用本应用即视为接受更新后的条款。

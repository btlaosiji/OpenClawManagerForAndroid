OpenClaw 管理 隐私政策

本政策说明 OpenClaw 管理（下称「本应用」）如何对待您的信息。

一、数据存储与处理

本应用在您设备本地处理与存储以下信息：

  • Gateway 连接信息：您填写的 Gateway 地址（URL）与 Token，在连接成功后会被保存到本机存储（如 SharedPreferences），仅用于下次启动时自动填充，不会上传至我们或任何第三方服务器。
  • 对话与配置：与 Gateway 的对话记录、模型选择、API Key 等仅在您设备与您指定的 Gateway 之间传输；本应用不会将上述内容发送到我们控制的服务器。

二、网络访问

本应用会发起网络请求的场景包括：

  • 连接 OpenClaw Gateway：您输入的 Gateway 地址（WebSocket），数据仅在您设备与您的 Gateway 之间传输。
  • Skill 搜索：调用第三方 Top ClawHub Skills API（如 topclawhubskills.com）以展示可安装的 Skill 列表与复制安装命令。该请求可能包含搜索关键词，请参考该第三方的隐私政策。

我们不会收集、存储或转发您的对话内容、API Key 或 Gateway Token 到本应用开发者控制的服务器。

三、第三方服务

使用 Skill 搜索功能时会访问第三方 API，其数据实践遵循该第三方政策。使用 Gateway 时，您配置的模型提供商（如 OpenAI、Gemini 等）的数据处理遵循各提供商政策。

四、政策更新

我们可能适时更新本隐私政策，更新版本将在应用内或发布渠道公布。继续使用本应用即视为接受更新后的政策。
